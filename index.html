General Machine Intelligent System

Welcome to GMIS

通用机器智能系统 （General Machine Intelligent System ，GMIS）是一个通用的机器人操作系统。它可以和用户进行类自然语言交互，并将用户说的话实时理解、编程为自身逻辑行为输出，从而满足用户的个性化需求。任何基于GMIS的智能硬件平台都将因此变成机器人，并可能在未来实现自主智能。更多的技术说明可以参考这里。

经过12年的研发，目前GMIS已经得到一个技术原型。这里是一个简单的原型的演示:视频(中文，18分钟)，如果不能直接打开，可以从这里下载下载。在另一个视频里，介绍了一下GMIS的设计哲学:视频(中文，45分钟)，同样可以在这里下载。如果你具备基本编程经验，也可以在这里下载试用原型。
